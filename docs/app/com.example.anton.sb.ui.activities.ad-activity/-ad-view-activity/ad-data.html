adData

private fun adData(adId: Long, title: TextView, city: TextView, description: TextView, price: TextView, username: TextView, telephone: TextView, photo: ImageView): Unit

Get ad information. This method use ApiService.getAd and processing response from server. If response is successful, then display information about ad, else - display error processing by handleError.

Parameters

adId - ad id

title - title ad

city - city of ad

description - description of ad

price - ad price

username - full name of owner ad

telephone - phone number of owner ad

photo - ad photo
